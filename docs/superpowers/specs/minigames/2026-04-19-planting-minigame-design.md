## 一、游戏概述

### 1.1 游戏定位

种植是药园生产流程的起点（种植→炮制→煎药），玩家通过选择种子、地块、水源、肥料，学习中药材种植的四气五味归经知识，实现"操作中记忆"的核心设计理念。

### 1.2 核心玩法

| 玩法要素 | 描述 |
|---------|------|
| **学习目标** | 掌握四气(寒热温凉)、五味(酸苦甘辛咸)、归经概念 |
| **操作方式** | 4阶段流程：选种子→选地块→选水源→选肥料→生长→收获考教 |
| **难度设计** | 初级(提示匹配)→中级(无提示)→高级(自定义种植) |
| **反馈机制** | 匹配评分(地块/水源/肥料)，收获考教答题 |

### 1.3 游戏时长

- 单次种植: 2-4分钟(不含等待)
- 生长等待: 2-5分钟(可跳过)
- 收获考教: 30-60秒

## 三、游戏流程设计

### 3.1 主流程(6阶段)

```
开始种植
    │
    ▼
阶段1: 选择种子
    │ 显示背包种子列表(使用ItemSlot组件)
    │ 显示种子属性(四气/五味/归经)
    │ 选择要种植的种子
    ▼
阶段2: 选择地块
    │ 显示药园地块网格(使用地块可视化组件)
    │ 显示地块归经属性
    │ 选择合适地块(匹配种子归经)
    ▼
阶段3: 选择水源
    │ 显示水源选项(温泉水/温水/井水/溪水/冰水)
    │ 显示四气对应(热/温/平/凉/寒)
    │ 选择匹配水源(匹配种子四气需求)
    ▼
阶段4: 选择肥料
    │ 显示肥料选项(辛味肥/甘味肥/苦味肥/酸味肥/咸味肥)
    │ 显示五味对应
    │ 选择匹配肥料(匹配种子五味需求)
    ▼
阶段5: 生长等待
    │ 显示种植动画(播种→浇水→施肥)
    │ 显示生长进度(使用ProgressBar组件)
    │ 可跳过等待或观看生长动画
    ▼
阶段6: 收获考教
    │ 显示考教题目(药材功效/主治)
    │ 答对后收获入库
    │ 答错可重试或查看解释
    ▼
阶段7: 结果展示
    │ 显示匹配评分(地块30+水源35+肥料35)
    │ 显示考教结果
    │ 显示药袋分类建议
    ▼
结束种植
```

### 3.2 四气五味匹配逻辑

```
种子属性:
    required_water: 'warm'   (需求: 温性水源)
    required_fertilizer: 'pungent' (需求: 辛味肥料)
    meridian: 'lung'         (归经: 肺经)

匹配计算:
    地块匹配: 地块归经 == 种子归经 OR 地块为'general' → +30分
    水源匹配: 水源四气 == 种子四气 OR 水源为'neutral' → +35分
    肥料匹配: 肥料五味 == 种子五味 → +35分

总分计算:
    plot_match: true → 30分
    water_match: true → 35分
    fertilizer_match: true → 35分
    total_score: 100分 (满分)
```

## 四、四气五味系统

### 4.1 四气(水源对应)

| 水源 | 四气 | 适合药材 |
|-----|-----|---------|
| 温泉水 | 热 | 附子、干姜 |
| 温水 | 温 | 麻黄、桂枝 |
| 井水 | 平 | 通用 |
| 溪水 | 凉 | 薄荷、菊花 |
| 冰水 | 寒 | 金银花、连翘 |

### 4.2 五味(肥料对应)

| 肥料 | 五味 | 适合药材 |
|-----|-----|---------|
| 辛味肥 | 辛 | 麻黄、细辛(发散) |
| 甘味肥 | 甘 | 甘草、黄芪(补益) |
| 苦味肥 | 苦 | 黄连、连翘(清热) |
| 酸味肥 | 酸 | 五味子、乌梅(收敛) |
| 咸味肥 | 咸 | 昆布、海藻(软坚) |

### 4.3 归经(地块对应)

| 地块归经 | 适合药材 |
|---------|---------|
| 肺经 | 麻黄、杏仁、金银花 |
| 心经 | 桂枝、连翘 |
| 肝经 | 柴胡、白芍 |
| 脾经 | 白术、山药 |
| 肾经 | 熟地、杜仲 |
| 通用 | 各类药材 |
