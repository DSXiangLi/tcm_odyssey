## 一、游戏概述

### 1.1 游戏定位

种植是药园生产流程的起点（种植→炮制→煎药），玩家通过选择种子、地块、水源、肥料，学习中药材种植的四气五味归经知识，实现"操作中记忆"的核心设计理念。

### 1.2 核心玩法

| 玩法要素 | 描述 |
|---------|------|
| **学习目标** | 掌握四气(寒热温凉)、五味(酸苦甘辛咸)概念 |
| **操作方式** | 4阶段流程：选种子→选地块→选水源→选肥料→生长→收获考教 |
| **难度设计** | 初级(提示匹配)→中级(无提示)→高级(自定义种植) |
| **反馈机制** | 匹配评分(水源/肥料)，收获考教答题 |

### 1.3 游戏时长

- 单次种植: 2-4分钟(不含等待)
- 生长等待: 2-5分钟(可跳过)
- 收获考教: 30-60秒

## 三、游戏流程设计

### 3.1 主流程(6阶段)

```
开始种植
    │
    ▼
阶段1: 选择种子
    │ 显示背包种子列表(使用ItemSlot组件)
    │ 显示种子属性(四气/五味)
    │ 选择要种植的种子
    ▼
阶段2: 选择地块
    │ 显示药园地块网格(简化：无归经限制)
    │ 选择任意空闲地块
    │ (地块仅为种植位置，不再匹配归经)
    ▼
阶段3: 选择水源
    │ 显示水源选项(温泉水/温水/井水/溪水/冰水)
    │ 显示四气对应(热/温/平/凉/寒)
    │ 选择匹配水源(匹配种子四气需求)
    ▼
阶段4: 选择肥料
    │ 显示肥料选项(辛味肥/甘味肥/苦味肥/酸味肥/咸味肥)
    │ 显示五味对应
    │ 选择匹配肥料(匹配种子五味需求)
    ▼
阶段5: 生长等待
    │ 显示种植动画(播种→浇水→施肥)
    │ 显示生长进度(使用ProgressBar组件)
    │ 可跳过等待或观看生长动画
    ▼
阶段6: 收获考教
    │ 显示考教题目(药材功效/主治)
    │ 答对后收获入库
    │ 答错可重试或查看解释
    ▼
阶段7: 结果展示
    │ 显示匹配评分(水源50+肥料50)
    │ 显示考教结果
    │ 显示药袋分类建议
    ▼
结束种植
```

### 3.2 四气五味匹配逻辑

```
种子属性:
    required_water: 'warm'   (需求: 温性水源)
    required_fertilizer: 'pungent' (需求: 辛味肥料)

匹配计算:
    水源匹配: 水源四气 == 种子四气 OR 水源为'neutral' → +50分
    肥料匹配: 肥料五味 == 种子五味 → +50分

总分计算:
    water_match: true → 50分
    fertilizer_match: true → 50分
    total_score: 100分 (满分)
```

## 六、考题系统设计（收获验证）

### 6.1 设计理念

考题系统是"操作中记忆"的核心验证环节，玩家在每次收获时需要回答一道考题，确保对所种植药材的知识有实际掌握，而非机械操作。

### 6.2 考题触发机制

| 触发条件 | 行为 |
|---------|------|
| 作物成熟 | 显示成熟图标，可点击收获 |
| 点击收获 | 弹出考题弹窗(QuizModal) |
| 答题正确 | 收获药材入库，显示成功动画 |
| 答题错误 | 显示解释，可重试或放弃 |

### 6.3 考题类型与内容

每次收获随机弹出**1道题**（单选题），题目范围涵盖中药学核心知识：

| 考题类型 | 占比 | 题目示例 | 考察知识点 |
|---------|------|---------|-----------|
| **性味** | 20% | "麻黄的性味是？" A.辛温 B.甘寒 C.酸平 D.苦凉 | 四气+五味组合 |
| **归经** | 20% | "麻黄归哪条经？" A.肺经 B.心经 C.肝经 D.脾经 | 药物归经归属 |
| **功效** | 25% | "麻黄的主要功效是？" A.发汗解表 B.补气健脾 C.养血安神 D.清热解毒 | 药物主要功效 |
| **主治** | 25% | "麻黄主治什么病症？" A.风寒感冒 B.脾胃虚弱 C.失眠多梦 D.热毒疮疡 | 药物主治病症 |
| **配伍禁忌** | 10% | "麻黄不宜与何药配伍？" A.附子 B.甘草 C.细辛 D.白芍 | 配伍禁忌知识 |

### 6.4 考题数据结构

```typescript
// 单道考题
interface QuizItem {
  id: string;
  type: 'nature_flavor' | 'meridian' | 'effect' | 'indication' | 'contraindication';
  question: string;
  options: string[];         // 4个选项
  correctAnswer: number;     // 正确选项索引(0-3)
  explanation: string;       // 答错后显示的解释
}

// 药材考题库
interface HerbQuizBank {
  herbId: string;            // 药材ID(如 'mahuang')
  herbName: string;          // 药材名称
  quizzes: QuizItem[];       // 该药材的所有考题
}

// 考题系统状态
interface QuizState {
  currentQuiz: QuizItem | null;
  selectedAnswer: number | null;
  isCorrect: boolean;
  attempts: number;          // 已尝试次数
  maxAttempts: 3;            // 最大尝试次数
}
```

| 结果 | 奖励 |
|-----|------|
| 首次答对 | 收获药材 + 经验值×1.5倍 |
| 第2次答对 | 收获药材 + 经验值×1.0倍 |
| 第3次答对 | 收获药材 + 经验值×0.5倍 |
| 3次全错 | 无法收获，药材留在地块(可再尝试) |
| 放弃收获 | 药材留在地块，下次可再尝试 |
