双模式说明

📋 任务模式

Agent拥有一套任务链，自动管理

  • 发布任务（根据玩家进度）
  • 更新任务状态
  • 判断任务完成
  • 给予奖励
示例：玩家完成诊治 → 青木先生自动判断完成 → 发放奖励 → 解锁下一个任务

💬 对话模式

与玩家自由对话，回答问题

  • 回答中医相关问题
  • 解释方剂原理
  • 闲聊增加好感
  • 回顾过往对话
示例：玩家问"桂枝汤和麻黄汤有什么区别？" → 青木先生详细解释

统一对话接口

// 玩家与NPC对话（统一接口）
POST /api/npc/chat 
{
  "npc_id": "qingmu",
  "player_input": "师父，我完成了王婶的诊治",
  "context": {
    "player_level": 3,
    "current_task": "task_002"
  }
}
// NPC自动判断：这是任务相关对话
{
  "mode": "task",
  "response": "不错！王婶的风寒已解，你用桂枝汤很对症。来，这是你的奖励。",
  "task_update": {
    "task_id": "task_002",
    "status": "completed",
    "reward": { "gold": 30, "exp": 15 }
  },
  "next_task_hint": "明日会有新病人，准备学习麻黄汤。"
}

任务模式对话

玩家：师父，桂枝汤和麻黄汤有什么区别？

青木：好问题！桂枝汤治风寒表虚，麻黄汤治风寒表实。打个比方，桂枝汤像是给虚弱的人添件衣裳，麻黄汤则是让壮实的人发汗驱寒...
