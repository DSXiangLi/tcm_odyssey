## 设计方案参考

### 方案B: 3D立体边框
```
1. 外层边框: 4px亮绿(#80a040)
2. 主背景: 深绿(#1a2e26)，alpha=1.0
3. 顶部/左侧高光: 2px亮绿(#90c070)
4. 底部/右侧阴影: 2px暗棕(#604020)
5. 8px外部阴影（可选）
```

### 内容宽度约束公式
```
文字maxWidth = 区域宽度 - 边框*2 - padding*2
```

### Task 5: InquiryUI 内部布局修复 - 对话区域

**文件**: `src/ui/InquiryUI.ts`

**问题**: 对话文字宽度480超出背景460

**修改内容**:
| 元素 | 当前 | 新值 |
|-----|------|------|
| 对话背景宽度 | 460 | 保持460 |
| 对话文字宽度 | 480 | **440**（460-边框4-padding16） |
| 对话文本位置 | -380 | -370（与背景左边界对齐） |

**计算公式**:
```
文字宽度 = 背景宽度 - 边框宽度*2 - padding*2
         = 460 - 2*2 - 16*2
         = 460 - 4 - 32
         = 424 ≈ 440（留安全边距）
```
