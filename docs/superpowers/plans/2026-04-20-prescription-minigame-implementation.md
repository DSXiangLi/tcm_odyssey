## 一、开发概述

### 1.1 改造目标

将选方游戏从"文字选项"模式改造为"方剂卡片选择+组成可视化"模式。

### 1.2 核心特色

| 特色 | 描述 |
|-----|------|
| 辨证结果展示 | 显示已判断证型作为参考 |
| 方剂卡片 | 显示组成药材小卡片(4-5个) |
| 组成预览 | 选中方剂显示组成预览 |
| 方剂加减 | 高级解锁后可加减药材 |
