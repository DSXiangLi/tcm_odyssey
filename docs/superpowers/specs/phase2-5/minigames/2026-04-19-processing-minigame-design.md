## 一、游戏概述

### 1.1 游戏定位

炮制是药园生产流程的核心环节（种植→炮制→煎药），玩家根据药材特性，选择合适的炮制方法、辅料，掌握中药材加工处理技术。

### 1.2 核心玩法

| 玩法要素 | 描述 |
|---------|------|
| **学习目标** | 掌握炒/炙/煅/蒸/煮五大炮制方法，了解辅料功效 |
| **操作方式** | 统一场景：左侧炮制灶台动画+右侧背包(药材+方法+辅料) |
| **难度设计** | 初级(提示方法)→中级(无提示)→高级(多方法可选) |
| **反馈机制** | 方法/辅料/时间三维评分，错误详情 |

### 1.3 游戏时长

- 单次炮制: 2-4分钟
- 完整流程: 3-5分钟

### 2.1 上游依赖

| 来源 | 数据依赖 | 说明 |
|-----|---------|------|
| 种植 | harvestHerbs | 收获药材来源 |

| 输出数据 | 目标 | 说明 |

| processedHerb | 背包 | 炮制后药材入库 |

```

     ▼                                  ▼
┌─────────────────────────────────────────────┐

│                                             │

     │
```

## 四、炮制方法系统

### 4.1 五大方法大类

| 大类 | 具体方法 | 适用场景 |
|-----|---------|---------|
| 炒(chao) | 清炒、麸炒、土炒 | 增强健脾、改变药性 |
| 炙(zhi) | 蜜炙、酒炙、醋炙、盐炙 | 增强润肺/活血/入肝/入肾 |
| 煅(duan) | 明煅、煅淬 | 矿物药处理、降低毒性 |
| 蒸(zheng) | 清蒸、酒蒸 | 增强补益、软化药材 |
| 煮(zhu) | 水煮、醋煮 | 降低毒性、增强功效 |

### 4.2 辅料功效

| 辅料 | 功效增强 | 适用药材示例 |
|-----|---------|-------------|
| 蜂蜜 | 润肺止咳 | 麻黄→蜜麻黄、甘草→蜜甘草 |
| 黄酒 | 活血通络 | 当归→酒当归、大黄→酒大黄 |
| 米醋 | 入肝止痛 | 柴胡→醋柴胡、香附→醋香附 |
| 盐水 | 入肾补肾 | 杜仲→盐杜仲、黄柏→盐黄柏 |

### 5.2 核心交互

| 区域 | 交互方式 | 说明 |
|-----|---------|------|
| 右侧药材列表 | 点击选中 | 选中后显示✓，左侧显示药材名 |
| 右侧方法选择 | 点击选择 | 方法大类分组显示，选中显示✓ |
| 右侧辅料选择 | 点击选择 | 与方法匹配的辅料选项 |
| 左侧灶台动画 | AI生图占位 | 240×300，铁锅+火焰动效 |
| 左侧药材变换 | AI图+动效 | 原始药材→闪烁动效→饮片 |
| 开始炮制 | 点击按钮 | 药材+方法+辅料齐全后可用 |
| 成品区域 | 点击收取 | 炮制完成后点击进背包 |

### 5.3 AI生图需求

| 素材 | 尺寸 | 描述 |
|-----|-----|------|
| 炮制灶台动画 | 240×300 | 大铁锅+炉灶+火焰，可静止或简单动效 |
| 原始药材图 | 60×60 | 各药材原形态(如甘草原药材) |
| 饮片图 | 60×60 | 各药材炮制后形态(如蜜炙甘草) |
| 辅料图标 | 60×60 | 蜂蜜/黄酒/米醋/盐水各一个 |
