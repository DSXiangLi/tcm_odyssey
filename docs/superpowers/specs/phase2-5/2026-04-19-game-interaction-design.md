# UI交互设计方案 - 游戏化交互

**版本**: v2.0
**日期**: 2026-04-19
**参考**: 《桃源深处有人家》交互设计
**风格**: PC端游戏风格，减少文字，增加视觉交互

---

## 一、核心设计原则

### 1.1 统一交互模式

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  ┌───────────────────────┐  ┌─────────────────────────┐ │
│  │                       │  │                         │ │
│  │    目标区域(左侧)      │  │    资源面板(右侧)        │ │
│  │                       │  │                         │ │
│  │  - 空槽位等待填充      │  │  - 背包物品格子          │ │
│  │  - 流程进度指示        │  │  - 可用资源卡片          │ │
│  │  - 当前状态展示        │  │  - 分类标签切换          │ │
│  │                       │  │                         │ │
│  └───────────────────────┘  └─────────────────────────┘ │
│                                                         │
│  交互方式:                                               │
│  1. 点击资源 → 点击槽位 (PC友好)                        │
│  2. 拖放资源 → 放入槽位 (可选)                          │
│  3. 双击资源 → 自动放入第一个空槽位                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 1.2 减少文字，增加视觉

| 元素 | 旧设计(文字为主) | 新设计(视觉为主) |
|-----|-----------------|-----------------|
| 选项 | "○ 浮脉 ○ 沉脉" | 脉象图标卡片(浮↑/沉↓) |
| 槽位 | "君臣佐使"文字标签 | 角色图标 + 颜色区分 |
| 辅料 | "蜜炙需要蜂蜜"文字 | 蜂蜜图标 + 辅料槽位 |
| 状态 | "已选择: 浮脉" | 槽位中显示选中图标 |
| 提示 | 长文字描述 | 简短标签 + 知识卡片按钮 |

### 1.3 即时视觉反馈

| 操作 | 即时反馈 |
|-----|---------|
| 选择资源 | 资源卡片发光/放大 |
| 放入槽位 | 槽位填充动画 + 满足提示 |
| 配对正确 | 绿色脉冲 + 音效 |
| 配对错误 | 红色闪烁 + 修正提示 |
| 完成阶段 | 进度指示器推进 |

## 四、统一槽位类型规范

> **核心原则**: 每种槽位类型在所有小游戏中使用统一的UI设计，确保视觉一致性和交互可预测性。

### 4.1 槽位类型分类

| 槽位类型 | 适用游戏 | 用途 | 统一标识 |
|---------|---------|------|---------|
| **CompatibilitySlot** | 煎药、选方 | 配伍放置(君臣佐使) | 角色图标+颜色 |
| **SynthesisSlot** | 炮制 | 合成输入(药材+辅料) | 需求类型标签 |
| **PlotSlot** | 种植 | 地块选择 | 归经颜色+网格 |
| **DiagnosisSlot** | 脉诊、舌诊、辨证 | 诊断结果展示 | 简洁卡片槽 |
| **TimelineSlot** | 煎药 | 煎药顺序 | 时间轴位置 |

```
│   │                             │  │
│   │    [时间标记]               │  │  ← 底部: │去沫│ 或 │加入│
│   │                             │  │

│                                     │

```

### 4.7 槽位状态统一规范

| 状态 | 所有槽位统一表现 |
|-----|-----------------|
| **空槽** | 凹陷效果 + 虚线边框 + 等待提示文字 |
| **选中待填** | 高亮边框(白色) + 脉冲动画 |
| **填充中** | 卡片飞入动画 + 槽位放大效果 |
| **填充完成** | 实线边框 + 显示内容 + 满足指示(底部绿条) |
| **填充正确(评分)** | 绿色脉冲 + 音效 |
| **填充错误(评分)** | 红色闪烁 + 修正提示 |
| **禁用** | 半透明 + 禁止图标 |

## 六、交互动画规范

### 5.1 卡片飞入动画

```typescript
interface FlyInAnimation {
  start: { x: number; y: number };  // 起点(背包格子)
  end: { x: number; y: number };    // 终点(目标槽位)
  duration: 300;                     // 持续时间(ms)
  easing: 'Power2';                  // 缓动函数
  scale: { from: 1, to: 0.8 };      // 缩放(飞入时缩小)
  onComplete: () => void;            // 完成回调
}
```
