### 6.3 测试步骤

**T-E001: 场景进入**
1. 启动游戏，进入小镇外景
2. 移动玩家至诊所门口，按上键进入
3. 在诊所场景按D键
4. 验证：场景切换至 DecoctionScene

**T-E002: 卷轴风格UI呈现**
1. 进入煎药场景后
2. 验证：界面顶部显示木轴边框
3. 验证：主标题"煎药"居中显示
4. 验证：左上角显示"杏林"印章
5. 验证：右下角显示"煎煮"印章

**T-E003-T-E006: 药材拖拽交互**
1. 验证右侧药材区显示药牌网格
2. 验证药牌显示像素图标、药名、属性、数量
3. 拖拽药牌至左侧炉灶区域
4. 验证拖拽过程中显示金色轨迹
5. 投入炉灶后显示"合"印章爆发效果

**T-E007: ESC退出**
1. 按ESC键
2. 验证返回诊所场景
