## 问题清单

### 1. 标题画面文字可读性问题
- **位置**: TitleScene
- **问题**: "最近存档"、"方向键"等文字看不清
- **根因猜测**: TEXT_SECONDARY颜色(#b0a090)与背景对比度不足
- **影响范围**: 用户无法清晰阅读操作提示

### 2. 存档选择弹窗无边框
- **位置**: SaveUI（标题画面点击"开始游戏"后）
- **问题**: 弹窗没有边框，取消按钮看不清
- **根因猜测**: 弹窗背景色与游戏背景色相近，缺乏视觉分隔
- **影响范围**: 用户无法识别弹窗边界

### 3. 新游戏确认弹窗无边框
- **位置**: TutorialUI（点击"新游戏"后）
- **问题**: 弹窗没有边框，引导文字看不清
- **根因猜测**: 同上，背景色冲突
- **影响范围**: 用户无法识别弹窗内容

### 4. 进度条与边框不对齐
- **位置**: TutorialUI（引导进度条）
- **问题**: 进度条长度和边框没有对齐
- **根因猜测**: 进度条宽度计算或位置偏移问题
- **影响范围**: 视觉不协调，影响专业感

### 5. 场景提示文字溢出
- **位置**: TownOutdoorScene（进入游戏后）
- **问题**: "这是百草镇"提示文字超出边框
- **根因猜测**: 文字宽度超出预设容器宽度
- **影响范围**: 文字截断或溢出，影响阅读
