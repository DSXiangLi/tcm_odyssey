# Phase 2.5 通用组件开发计划

**版本**: v1.0
**日期**: 2026-04-20
**状态**: 待执行
**前置依赖**: BaseUIComponent, ModalUI, UIBorderStyles (已完成)

---

## 一、开发概述

### 1.1 目标

创建游戏交互改造所需的通用组件，为7个小游戏提供统一的基础设施。

### 1.2 组件清单

| 组件 | 用途 | 使用场景 |
|-----|------|---------|
| InfoCardComponent | 信息卡片弹窗 | 所有小游戏的知识卡片展示 |
| ProgressBarComponent | 进度条展示 | 煎药/炮制/种植进度 |
| FlyInAnimation | 卡片飞入动画 | 拖放交互时的视觉反馈 |
| ResourcePanel | 资源面板组件 | 煎药/炮制/种植的背包展示 |

### 1.3 开发顺序

```
Task 0: InfoCardComponent (知识卡片)
    │
    ▼
Task 1: ProgressBarComponent (进度条)
    │
    ▼
Task 2: FlyInAnimation (飞入动画)
    │
    ▼
Task 3: ResourcePanel (资源面板)
    │
    ▼
Task 4: 单元测试 + 集成测试
```

### 3.3 数据结构

```typescript
interface ProgressBarConfig {
  width: number;                // 进度条宽度
  height: number;               // 进度条高度(默认20)
  maxValue: number;             // 最大值(100或时间秒数)
  currentValue: number;         // 当前值
  showPercentage: boolean;      // 显示百分比
  showTime: boolean;            // 显示时间
  gradientColors: {             // 渐变颜色
    low: number;                // 低值颜色(红色)
    mid: number;                // 中值颜色(黄色)
    high: number;               // 高值颜色(绿色)
  };
  onComplete?: () => void;      // 完成回调
}

// 状态枚举
type ProgressBarState = 'empty' | 'filling' | 'complete' | 'error';
```

|-----|------|

| 缓动 | Power2缓动函数 |
| 完成回调 | 飞入完成触发 |

### 4.2 动画参数

```typescript
interface FlyInAnimationConfig {
  start: { x: number; y: number };  // 起点(背包格子)
  end: { x: number; y: number };    // 终点(目标槽位)
  duration: number;                  // 持续时间(默认300ms)
  easing: string;                    // 缓动函数(默认'Power2')
  scale: { from: number; to: number }; // 缩放(默认{1.0, 0.8})
  fadeOut: boolean;                  // 是否淡出(默认false)
  onComplete?: () => void;           // 完成回调
}
```

interface ResourceItem {
