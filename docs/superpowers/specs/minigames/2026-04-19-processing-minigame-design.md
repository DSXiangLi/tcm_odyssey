## 一、游戏概述

### 1.1 游戏定位

炮制是药园生产流程的核心环节（种植→炮制→煎药），玩家根据药材特性，选择合适的炮制方法、辅料，掌握中药材加工处理技术。

### 1.2 核心玩法

| 玩法要素 | 描述 |
|---------|------|
| **学习目标** | 掌握炒/炙/煅/蒸/煮五大炮制方法，了解辅料功效 |
| **操作方式** | 4阶段流程：选药→选方法→选辅料→炮制操作 |
| **难度设计** | 初级(提示方法)→中级(无提示)→高级(多方法可选) |
| **反馈机制** | 方法/辅料/时间/质量四维评分，错误详情 |

### 1.3 游戏时长

- 单次炮制: 2-4分钟
- 完整流程: 3-5分钟

## 二、与其他游戏关联

### 2.1 上游依赖

| 来源 | 数据依赖 | 说明 |
|-----|---------|------|
| 种植 | harvestHerbs | 收获药材来源 |
| 背包 | herbs库存 | 可炮制药材 |
| 经验值 | unlockLevel | 炮制方法解锁 |

### 2.2 下游输出

| 输出数据 | 目标 | 说明 |
|---------|-----|------|
| processedHerb | 背包 | 炮制后药材入库 |
| score | 经验值 | 获得炮制经验 |
| feedback | NPC | 老张评述 |

### 2.3 数据流向图

```
种植(Planting) + 背包               炮制参数数据

     ▼                                  ▼
┌─────────────────────────────────────────────┐
│              炮制 (ProcessingScene)          │
│                                             │
│  阶段: 选药→选方法→选辅料→预处理→炮制         │
│  输入: herbs + unlock                       │

     │

背包(Inventory) → 煎药(Decoction)
```

## 三、游戏流程设计

### 3.1 主流程(5阶段)

```
开始炮制
    │
    ▼
阶段1: 选择药材
    │ 显示可炮制药材列表(使用ItemSlot组件)
    │ 显示药材炮制要求提示
    │ 选择要炮制的药材
    ▼
阶段2: 选择方法
    │ 显示方法大类(炒/炙/煅/蒸/煮)
    │ 显示具体方法选项(清炒/麸炒/蜜炙...)
    │ 检查方法与药材匹配
    ▼
阶段3: 选择辅料(可选)
    │ 如果方法需要辅料，显示辅料选项
    │ 显示辅料功效说明
    │ 选择合适的辅料
    ▼
阶段4: 预处理(简化)
    │ 显示预处理步骤(清洗/浸泡/切片)
    │ 简化为确认步骤
    │ 显示预处理要点
    ▼
阶段5: 炝制操作
    │ 显示炮制进度(使用ProgressBar组件)
    │ 显示炮制动画(炒/炙/煅视觉效果)
    │ 可中途检查质量
    │ 判断炮制终点
    ▼
阶段6: 评分展示
    │ 显示各维度评分
    │ 显示炮制前后对比
    │ 显示反馈建议
    ▼
结束炮制
```

### 3.2 炮制方法选择流程

```
显示方法大类卡片
    │
    │ ┌─────────┐ ┌─────────┐ ┌─────────┐
    │ │   炒    │ │   炙    │ │   煅    │ ...
    │ │ 清炒    │ │ 蜜炙    │ │ 明煅    │
    │ │ 麸炒    │ │ 酒炙    │ │ 煅淬    │
    │ │ 土炒    │ │ 醋炙    │ │         │
    │ └─────────┘ └─────────┘ └─────────┘
    │
    ▼
点击大类展开具体方法
    │
    ▼
选择具体方法
    │
    ├── 显示方法详情
    │   ├── 温度范围
    │   ├── 时间范围
    │   ├── 是否需要辅料
    │   └── 视觉效果描述
    │
    ▼
确认方法选择
```

## 四、炮制方法系统

### 4.1 五大方法大类

| 大类 | 具体方法 | 适用场景 |
|-----|---------|---------|
| 炒(chao) | 清炒、麸炒、土炒 | 增强健脾、改变药性 |
| 炙(zhi) | 蜜炙、酒炙、醋炙、盐炙 | 增强润肺/活血/入肝/入肾 |
| 煅(duan) | 明煅、煅淬 | 矿物药处理、降低毒性 |
| 蒸(zheng) | 清蒸、酒蒸 | 增强补益、软化药材 |
| 煮(zhu) | 水煮、醋煮 | 降低毒性、增强功效 |

### 4.2 辅料功效

| 辅料 | 功效增强 | 适用药材示例 |
|-----|---------|-------------|
| 蜂蜜 | 润肺止咳 | 麻黄→蜜麻黄、甘草→蜜甘草 |
| 黄酒 | 活血通络 | 当归→酒当归、大黄→酒大黄 |
| 米醋 | 入肝止痛 | 柴胡→醋柴胡、香附→醋香附 |
| 盐水 | 入肾补肾 | 杜仲→盐杜仲、黄柏→盐黄柏 |
